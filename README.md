# LPIC_Practice

- 学習用VMは、Dockerディレクトリ内にある、docker-compose.ymlにOSのimageが記載されている
  - Dockerディレクトリ内で以下のコマンドを実行し、コンテナにログインする
    ```
    $ docker-compose up -d

    $ docker container exec -it docker-linux-1 bash
    ```
